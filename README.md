# 美妆种草清单助手（H5）

这是一个单文件移动端 H5 演示应用（HTML/CSS/原生 JS + localStorage）。

## 部署到 GitHub Pages（获得可分享链接）

1. 把以下文件上传到仓库根目录（与 README.md 同级）：
   - `index.html`
   - `花知晓提亮液.jpg`
   - `毛戈平粉底.jpg`
   - `腮红丝字.jpg`

2. 进入 GitHub 仓库：**Settings → Pages**
   - **Build and deployment** 选择 **Deploy from a branch**
   - **Branch** 选择 `main`，目录选择 `/ (root)`
   - 点击 **Save**

3. 等待 Pages 构建完成后，访问：

`https://QuinnMercer.github.io/beauty-wishlist-h5/`

## 本地运行

直接双击打开 `index.html`，或用任意静态服务器启动：

```bash
python -m http.server 5173
```

然后打开：`http://localhost:5173/`
