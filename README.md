# 美妆种草清单助手（H5）

单文件移动端 H5 演示应用（HTML/CSS/原生 JS + localStorage）。

## 你需要上传哪些文件

把以下文件放到仓库根目录（与 README.md 同级）：

- `index.html`
- `清单.jpg`（添加页的“清单样例”会用到）

> 仓库里如果还有历史测试图片（如 `花知晓提亮液.jpg` 等）不影响运行。

## 部署到 GitHub Pages（获得可分享链接）

1) 上传最新文件

- 打开仓库：`https://github.com/QuinnMercer/beauty-wishlist-h5`
- 点击 **Add file → Upload files**
- 选择并上传：`index.html`、`清单.jpg`
- 下拉到页面底部点击 **Commit changes**

2) 开启 Pages

- 进入 **Settings → Pages**
- **Build and deployment** 选择 **Deploy from a branch**
- **Branch** 选择 `main`，目录选择 `/ (root)`
- 点击 **Save**

3) 访问链接

等待 Pages 构建完成后访问：

`https://QuinnMercer.github.io/beauty-wishlist-h5/`

## 常见问题

- **访问 404**：Pages 开启后需要等待一会儿；也检查 `Branch` 是否选了 `main / (root)`。
- **页面没更新**：浏览器强制刷新（Ctrl+F5）或等 CDN 缓存；也确认你上传的是最新的 `index.html`。
